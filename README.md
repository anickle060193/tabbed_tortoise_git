# Tabbed TortoiseGit

A simple tabbed wrapper for [TortoiseGit](https://tortoisegit.org/) to host multiple TortoiseGit logs within a single application.


### Installation
Download and install [Setup.msi](https://github.com/anickle060193/tabbed_tortoise_git/raw/0.12.0/Setup/Output/Setup.msi).
